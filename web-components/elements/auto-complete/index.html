This web-component to replace the existing jQuery auto-complete component. Using the native HTML5 datalist and built-in browser capabilities to parse through existing lists.

Default included information is {% raw %}{{ name }}{% endraw %} {% raw %}{{ sector }}{% endraw %} and a label attribute with {% raw %}{{ state }}{% endraw %}.

A few test examples:

  • Time Warner Cable
  • Texas Tech
  • Life Time Fitness
  • Tyco International
  • Hankore Environment Tech
